## Задание  Л2.8
Решение задания находится в файле ['l2-8.go'](./l2-8.go)
Команды для запуска и тестирования решения:

1. Запуск приложения:

'''bash
task run
'''

2. Компиляция исполняемого файла:

'''bash
task build
'''

3. Запуск тестов(golangci-lint, vet):

'''bash
task check
'''
